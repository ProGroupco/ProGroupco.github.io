// Examples

See It In Action

// Your first LEL program
Say "Hello, World!";
// Define a function
Make Function greet(name) {
  Say "Hello, " + name;
}
 
// Call it
Use greet("World");
// Loop through numbers
Make i = 1;
While i <= 5 {
  Say i;
  i = i + 1;
}
LEL vs Python — same logic, different feel
LEL
Make Function add(a, b) {
  Give Back a + b;
}
Make result = add(10, 5);
Say result;
Python equivalent
def add(a, b):
  return a + b
 
result = add(10, 5)
print(result)
